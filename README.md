# zipfile-fusions-cartoradio
URL : https://ed94120.github.io/zipfile-fusion-cartoradio/

L'utilitaire lit les fichiers ZIP ANFR contenant les mesures d'Expositions extraites du site ANFR Cartoradio.

## Objectif
Le fichier contenant les métadonnées et la mesure Cas A est fusionné avec le fichier contenant les composantes spectrales de la mesure Cas B.

Un fichier CSV est crée

## Utilisation
Le fichier ANFR ZIP est déposé par l'utilisateur dans un champ prévu à cet effet.

Plusieurs fichiers ZIP sont disponibles pour être utilisés comme exemples.
